# Supply Chain Delivery Performance Analysis

An end-to-end Supply Chain Analytics project using Python, Machine Learning, and Power BI to analyze delivery performance across **180,520 e-commerce orders**.

## Business Problem

Actual shipping times frequently deviate from scheduled delivery windows, leading to reduced customer trust, lower order profitability, and unreliable delivery commitments.

## Key Highlights

* Analyzed **180,520 orders** from a global e-commerce supply chain.
* Identified a **54.71% late delivery rate**.
* Quantified **$2.1M profit at risk** due to delayed orders.
* Built a **Random Forest model** with **74% accuracy** to predict late deliveries.
* Developed an interactive **Power BI dashboard** for executive reporting.
